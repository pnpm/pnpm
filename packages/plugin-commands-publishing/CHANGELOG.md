### Minor Changes

- b734b45ea: By default, for portability reasons, no files except those listed in the bin field will be marked as executable in the resulting package archive.  
  ```json
  "publishConfig": {
    "executableFiles": [
      "./dist/shim.js",
    ]
  }
  ```
