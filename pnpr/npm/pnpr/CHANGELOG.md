# @pnpm/pnpr

## 0.1.0

### Minor Changes

- `@pnpm/pnpr` is now versioned with changesets and released from the unified release workflow. Versions switch from the `0.0.0-<datestamp>` scheme to semver, starting at 0.1.0.
- The install script exits without error in the pnpm monorepo checkout, where the per-platform binary packages are not generated.
