# pax Documentation

pax is a universal package manager built as a fork of [pnpm](https://github.com/pnpm/pnpm). It extends pnpm with cross-ecosystem interoperability, TypeScript-first package configuration, and agentic workflows.

- [Architecture](./architecture.md) — How pax is structured and how it extends pnpm.
- [pnpm Compatibility](./pnpm-compatibility.md) — Tracks all departures from upstream pnpm behavior.
- [TypeScript Configuration](./typescript-config.md) — The `package.config.ts` design and API.
